Greetings fellow builder! Welcome to the blog for Builderment. Here you’ll find the latest updates for the game and previews of features still in development.

Early concept video of the game style

What is Builderment?

Builderment is a mobile game centered around building and automation. You’ll need to build machines to mine resources and setup production lines to craft those into more and more advanced items. Along the way you’ll be able to unlock new technologies that give you access to smarter and more efficient machines. You’ll also need to build strategically to survive, you won’t be alone.

When will it be available?

I’m hoping to open up alpha testing later this year but it’s still very early in development so dates are very much the enemy. All updates will be posted here and shared on twitter and instagram so subscribe and follow Builderment on those platforms for the latest info.
