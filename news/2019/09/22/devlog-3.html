This week life was pretty busy but I made some good progress on Builderment. Early in the week I added some life to the world by animating the water tiles. This was done by randomly rotating tiles every few seconds. I think the results are pretty good for how little effort was involved. üëè

for tile in visibleWaterTiles {
    // Target fps is 60 so on average each tile will rotate every 2 seconds
    if Int.random(in: 0 ..< 120) == 0 {
        // Select random rotation in 90 degree increments
        let rotation = Float(Int.random(in: 0 ..< 4)) * Float.pi / 2
        tile.node?.eulerAngles.y = rotation
    }
}
